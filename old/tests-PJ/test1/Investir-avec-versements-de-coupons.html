@njit(nogil=True)
def compute_mu(m, sigma, S, set_seed = True):
    if set_seed: np.random.seed(0)
    mu = np.empty(S)
    for s in np.arange(S):
        mu[s] = m + sigma*np.random.normal()
    return mu

  • Une fonction qui permet de calculer la valeur actualise de la richesse du porteur en $k=1$, $\tilde{\mathcal W}$. Pour mmoire, on a

    $\tilde{\mathcal W} = \mathcal{W}_0 + \frac{\mu - r}{1 + r}\left( \mathcal{W}_0 - c \right)\phi - c.$

    Cette fonction prend en entre les paramtres $\mathcal{W}_0$, $r$, $\mu$, $\phi$ et $c$ et renvoie en sortie une matrice dont les lignes sont des ralisations indpendantes du processus des richesses (valeurs actualises zro). Les variables $c$ et $\phi$ sont contraintes dans $[0, \mathcal{W}_0]$ et $[0, 1]$ respectivement.

@njit(nogil = True)
def compute_actualized_wealth_one_period(W0, r, mu, phi, c):
    S, = mu.shape
    W = np.zeros(S)
    for s in np.arange(S):
        W[s] = W0 + (mu[s] - r)*(W0 - c)*phi/(1 + r) - c
    return W

  • Une fonction qui calcule $\phi^*(c)$, la proportion d'actif risqu investie sur le montant restant aprs la perception du coupon. Pour rappel, $\phi^*(c) = \min\left( 1, \frac{1 + r}{\sigma\left( 1.96 - s \right)} \frac{\rho\mathcal{W}_0}{\mathcal{W}_0 - c} \right)$ avec $s = (m - r)/\sigma$.

    Elle prend en entre $\mathcal{W}_0$, $\rho$, $r$, $m$, $\sigma$ et un vecteur de valeurs $c$ et rend en sortie un vecteur de valeurs $\phi*$ correspondant.

@njit(nogil = True)
def compute_phi_star(W0, rho, r, m, sigma, c):
    N_c = c.shape[0] - 1
    phi_star = np.ones(N_c + 1)
    sharpe = (m - r)/sigma
    constant = (1 + r)/(sigma*(1.96 - sharpe))*rho*W0
    W0_over_N_c = W0/N_c
    for n_c in np.arange(N_c):
        c = W0_over_N_c*n_c
        value = constant/(W0 - c)
        if value >= 1: continue
        phi_star[n_c] = value
    return phi_star

N_c = 100
c = np.arange(0, W0, 1/(N_c + 1))
phi_star = compute_phi_star(W0, rho, r, m, sigma, c)
c_star = W0*(1 - (1 + r)/(sigma*(1.96 - sharpe))*rho)

Quelques remarques.¶

  • Comme on peut le voir sur le trac ci-dessus (excuter la cellule si ce n'est pas encore fait), passer une certaine valeur $c^*$, il devrait devenir sous-optimal de prendre plus de capital.

    La valeur $c^*$ est obtenue en galisant $1$ l'expression $\frac{1 + r}{\sigma\left( 1.96 - s \right)} \frac{\rho\mathcal{W}_0}{\mathcal{W}_0 - c}$, ce qui conduit, sans trop d'effort

    $c^* = \mathcal{W}_0\left( 1 - \frac{1 + r}{\sigma\left( 1.96 - s \right)} \rho \right)$.

    Au del de cette valeur, la portion de capital allou l'actif risqu est cape car on devrait dpasser le montant disponible investir. Les gains obtenus vont donc diminuer au fir et mesure que l'on se verse un coupon plus important.

  • Dans le cas o l'on ne se verserait pas de coupon, la part $\phi_{\min{}}^*$ dans l'actif risqu vaut

    $\phi_{\min{}}^* = \frac{1 + r}{\sigma\left( 1.96 - s \right)}\rho$.

  • Pour caractriser la situation o quoiqu'il arrive le porteur doit tout mettre dans l'actif risqu, galisons $\phi_{\min{}}^*$ 1. On trouve une valeur maximale pour $\rho$ gale

    $\rho_{\max{}}^* = \frac{\sigma\left( 1.96 - s \right)}{1 + r}$.

    Cette grandeur dfinit la proportion au dessus de laquelle, le porteur, pour maximiser ses gains, doit tout mettre dans l'actif risqu.

Analyse de la performance¶

Pour analyser la performance de la stratgie qui consiste placer la part $\phi^*(c)$ dans l'actif risqu sur la montant restant aprs versement du coupon $c$, on va utiliser une mthode de Monte-Carlo.

Quelques remarques¶

Comme on peut le voir sur la figure ci-dessus (excuter la cellule si ce n'est pas encore fait), au del d'un versement de $c^*$, le porteur perd en possibilit de rendements dans l'actif sans risques.

Quand le coupon qu'il se verse dpasse $c^*$, tout le montant restant doit tre allou l'actif sans risque, lui assurant gain moyen sous-optimal $\frac{\mu - r}{1 + r}\left( \mathcal{W}_0 - c \right)$ qui dcrot avec l'augmentation des versements en coupon!

Enfin, puisque les gains sont les mmes jusqu'en $c^*$, d'un point de vue purement de l'investissement financier, il est optimal de se verser le coupon $c^*$.

Application de cette stratégie avec répétition¶

Le paragraphe prcdent a montr comment le porteur pouvait choisir son coupon pour un investissement en deux priodes. Supposons qu'il garde la mme approche entre deux instants de rebalancement: il ne souhait pas, d'une priode l'autre voir plus d'une fraction $\rho$ de son capital disparatre.

Quelques notations et équations.¶

  • La richesse actualise en zro du porteur l'instant $k$ s'crit $\tilde{\mathcal{W}}_k$. Entre $k$ et $k+1$ cette richesse connat une variation

    $\tilde{\mathcal{W}}_{k+1} - \tilde{\mathcal{W}}_k = \left[ \tilde{\mathcal{W}}_k - \min\left( \frac{c}{\left( 1 + r \right)^{k + 1}}, \tilde{\mathcal{W}}_k \right) \right] \phi^*_k(c) \frac{\mu_k - r}{1 + r} - \frac{c}{\left( 1 + r \right)^{k + 1}}$.

    Le coupon doit tre actualis car il est considr comme fixe chaque priode de temps (un peu comme un bond). Aussi, on ajoute une borne au coupon effectivement peru si la richesse l'instant $k$ ne permet pas de le verser intgralement. Dans un tel cas, on distribue l'entiret de la richesse cet instant.

  • La stratgie d'investissement $\phi^*_k(c)$ est la mme chaque priode. On notera une dpendance en le temps provenant de la variation de richesse disposition au cours de l'investissement.

    $\phi^*_k(c) = \min\left( 1, \frac{1 + r}{\sigma\left( 1.96 - s \right)} \frac{\rho\tilde{\mathcal{W}}_k}{\tilde{\mathcal{W}}_k - c/\left( 1 + r \right)^{k + 1}} \right)$

    Elle assure au porteur qu'entre chaque priode, la probabilit qu'il perde une fraction $\rho$ de son capital l'instant $k$ ne dpasse pas 5%.

  • Le gain pour l'investisseur en $K$ est la somme des revenus qu'il a russi obtenir des ses placements risqu. Ce que l'on notera

    $\tilde{\mathcal{G}}_{K} = \sum_{k = 0}^{K - 1}\left( \tilde{\mathcal{W}}_k - \frac{c}{\left( 1 + r \right)^{k}} \right) \phi^*_k(c) \frac{\mu_k - r}{1 + r} $

Cherchons comment ce gain ce comporte avec une mthode de Monte-Carlo.

On supposera que l'investisseur une horizon de temps $K$ qui sera prcis par la suite.

@njit(nogil = True)
def compute_investment_data(c, W0, rho, r, m, sigma, K, set_seed = False):
    mu = compute_mu(m, sigma, K, set_seed = set_seed)
    phi = np.zeros(K)
    received_coupons = np.zeros(K)
    W = np.zeros(K + 1)
    G = np.zeros(K + 1)
    W[0] = W0
    G[0] = W0
    for k in np.arange(K):
        W_k = W[k]
        c_k = c/np.power(1 + r, k)
        if W_k <= c_k:
            received_coupon_k = W_k
            phi_k = 0               # Cette valeur n'est pas importante donc on la fixe volontairement  0
        else:
            received_coupon_k = c_k
            phi_k = np.minimum(1, (1 + r)/(sigma*(1.96 - sharpe))*rho*W_k/(W_k - c_k))
        psi_k = (W_k - c_k)*phi_k
        Delta_G_K = psi_k*(mu[k] - r)/(1 + r)
        G[k + 1] = G[k] + Delta_G_K
        W[k + 1] = np.maximum(W_k - c_k + Delta_G_K, 0)
        received_coupons[k] = received_coupon_k
        phi[k] = phi_k
    return phi, received_coupons, W, G

@njit(nogil = True)
def compute_investment_data_Monte_Carlo(c, W0, rho, r, m, sigma, K, S):
    phi = np.empty((S, K))
    received_coupons = np.empty((S, K))
    W = np.empty((S, K + 1))
    G = np.empty((S, K + 1))
    set_seed = False
    for s in np.arange(S):
        if s == 0: 
            set_seed = True
        else:
            set_seed = False
        phi[s, :], received_coupons[s, :], W[s, :], G[s, :] = compute_investment_data(c, W0, rho, r, m, sigma, K, set_seed = set_seed)
    return phi, received_coupons, W, G

@njit(nogil = True)
def compute_performance(coupons, W0, rho, r, m, sigma, K, S):
    probability_negative = np.zeros_like(coupons)
    Delta_probability_negative = np.zeros_like(coupons)
    G_star_K = np.zeros_like(coupons)
    Delta_G_star_K = np.zeros_like(coupons)
    for n_c in np.arange(coupons.shape[0]):
        c = coupons[n_c]
        phi, received_coupons, W, G = compute_investment_data_Monte_Carlo(c, W0, rho, r, m, sigma, K, S)
        probability_negative[n_c] = (G[:, -1] < (1 - rho*W0)).mean()
        Delta_probability_negative[n_c] = 1.96/np.sqrt(S)*(G[:, -1] < (1 - rho*W0)).std()
        G_star_K[n_c] = G[:, -1].mean()
        Delta_G_star_K[n_c] = G[:, -1].std()
    return probability_negative, Delta_probability_negative, G_star_K, Delta_G_star_K
